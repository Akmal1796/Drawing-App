Welcome to my drawing app. Below, you can see the steps to use this software easily. Feel free to see all the instructions for a better experience with the software.

You can use this software without having an account with limited features. The guest version of my software also provides better features to its users. For example, guest users can also save their works as images on their PCs.

Here you can see the extra features like save projects, open projects, and my projects. Each of these features provides the user with a better experience when using this software. You can save your work on the cloud and access it anytime. This will prevent data loss when you lose your work on your PC.

Here you see the simple login page. A user can login if they already registered.

If the user does not have an account, they can register via a registration form.

Here is the registration form. It just takes one minute to register.

Every type of user can save their work as an image to their PC by clicking the save as an image button.

You can use shapes that are available in the menu bar to draw structured drawings. You can also use a free brush that will help you draw on the screen, like drawing on paper. You can adjust the size of the brush as well.

You can use an eraser to erase the drawing, just like painting, and if you want to clear all the drawings, you can simply click the clear canvas button, and it will clear all the drawings on the canvas. You can adjust the size of the eraser as well.

You can save your work by simply clicking the save project button. It will save your work in our database.
